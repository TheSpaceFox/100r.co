{Uxn} is a portable virtual machine, meaning that it is a computer designed especially to be ported to different consoles and devices. The same Uxn program can be used on a NintendoDS, on a desktop computer, in a browser, or on tiny electronics - without doing any changes to the program itself.

Get Started

This guide will focus on getting you started using Uxn on a desktop computer, but if you quickly want to try Uxn, you can load a rom directly in your browser by using the web version of Uxn.

The desktop emulator is built with SDL2, download SDL2 for your operating system. For example, if you are using Ubuntu, use the following commands to install SDL2: 

sudo apt install libsdl2-dev

The next step is for you to download the 

git clone https://git.sr.ht/~rabbits/uxn
cd uxn
./build.sh
