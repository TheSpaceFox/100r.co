A drawing program

Noodle is currently used in the creation of various monochromatic illustration projects such as the Neauismetica, it shares a lot of the functionalities of the spritesheet editor {Nasu}.

Noodle is written in the Uxntal programming language and is compiled with the {Uxn} assembler, it features a handful of patterns, a pixel-perfect clean up brush, zoom functionalities and can export and import bit maps. We will write a detailed guide for this tool very soon.

Try running Noodle on the Uxn Emulator on Nintendo DS.

Getting Started

Noodle is distributed as a rom for the {Uxn} system, you must have an emulator available for your machine, for more details, see the {Uxn Guide}. To begin, navigate to the folder in which you want your project files to be saved, and enter the following to launch the noodle.rom:

./path/to/uxnemu ~/roms/noodle.rom

Controls

  • right-click erase
  • arrows move zoom
  • space toggle zoom
  • tab invert image
  • backspace blank canvas
  • 1-8 select brush size
  • ctrl+n clear
  • ctrl+r rename
  • ctrl+o load
  • ctrl+s save

Re-naming a file

The default filename reads untitled.bit. To change it, click on the text. The text field will blink, allowing you to type a new file name.

Invert images

To invert the image, press tab.

Saving file

To save a file, click on the black star located on the bottom right of the window.

The star has two states: The black state indicates that the file was saved, the red state indicates that the file has unsaved changes.

Loading file

To load an file, replace the active filename and write the name of the .chr file you wish to import, and click enter. To import the file click on the checkered square on the bottom right of the window.

If switching between projects, you can have multiple instances of the Uxn emulator open at various locations. Holding the alt key while clicking on the load-icon, will load the file as a 1bpp spritesheet.

New file

To load a blank file, click on the empty square icon on the bottom right of the window.

All changes will be lost, make sure to re-name your file.

Focus view

Click on the teal band around the window to switch to focus view. This will make the menu disappear. Click again to bring the menu back.

Resize canvas

To resize the canvas, pull on the black corner on the bottom right of the window. Adjusting the canvas size changes the number on top, next to the pattern menu.

Only resize the canvas at the start of a project.

Pattern menu

A set of 8 patterns to draw with.

The default pattern is black. Select the brush, or rectangle tool first, and then a pattern. It is possible to erase with patterns by right-clicking.

Pen

A tool to draw thin single-pixel drawings on the canvas.

This tool can only ever be 1 pixel. Right-click to erase using the pen tool.

Brush tool

A tool to draw brush-like strokes on the canvas.

To draw using a pattern, select a style from the pattern menu.

Line tool

A tool to draw straight lines on the canvas.

Lines will always be 1 pixel wide. Right-click to erase using the line tool.

Magic tool

A tool to clean up lines.

This tool is useful to remove excess pixels when drawing sharp edges with the pen tool.

Rectangle tool

A tool to draw rectangles on a canvas.

To draw using a pattern, select a style from the right menu.

Zoom tool

For a more detailed view of a drawing, select the magnifying glass, or press on the spacebar. The magnifying glass has a little circle next to it when zoomed in.

Use the arrows to move while zoomed in, and press the spacebar to return to the normal view.
