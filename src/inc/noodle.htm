A drawing program

Noodle is currently used in the creation of various monochromatic illustration projects such as the Neauismetica, it shares a lot of the functionalities of the spritesheet editor {Nasu}.

Noodle is written in the Uxntal programming language and is compiled with the {Uxn} assembler, it features a handful of patterns, a pixel-perfect cleanup brush, zoom functionalities and can export and import bit maps. We will write a detailed guide for this tool very soon.

Try running Noodle on the Uxn Emulator on Nintendo DS.

Controls

  • right-click erase
  • arrows move zoom
  • space toggle zoom
  • tab invert image
  • backspace blank canvas
  • 1-8 select brush size
  • ctrl+n clear
  • ctrl+r rename
  • ctrl+o load
  • ctrl+s save
