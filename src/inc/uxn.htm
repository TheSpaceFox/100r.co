The Uxn/Varvara ecosystem is a personal computing stack based on a small virtual machine, created to host our tools and games, programmable in its own unique assembly language.

It was designed with an implementation-first mindset with a focus on hosting graphical applications, the distribution of Uxn projects is akin to sharing game roms for any classic console emulator, to learn more, see our notes on {uxn design}.

Desktop Versions

To run Uxn programs, you'll need an emulator that works on your operating system. Below are links to emulators for major systems, and for a more obscure system. Alternatively, you can also implement your own by looking a the VM specs, the IO specs and the source code.

Linux Windows MacOS
